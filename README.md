#### Production work

**Containers & orchestration** — Right-sized production workloads from k3s Kubernetes to Docker Compose to cut operational overhead for a small team · `nginx-proxy` reverse proxy with automatic Let's Encrypt SSL across all domains

**CI/CD** — GitHub Actions → GHCR → SSH deploy, with zero-downtime container swaps and health checks

**Observability** — Prometheus · Grafana · cAdvisor · node-exporter, with high-memory and container-down alert rules wired to email · Portainer for container ops

**Reliability & security** — Nightly PostgreSQL backups, 7-day retention · Closed publicly exposed Postgres ports across 4 projects via iptables `DOCKER-USER` rules after a Hetzner/BSI alert (Docker bypasses UFW — `DOCKER-USER` is the fix)

**Mixed environments** — Windows Server 2025 on Hetzner dedicated hardware (RDP/NLA, hardened) · Cloudflare DNS for an AWS EKS cluster · Microsoft 365 / Azure AD SSO + tenant admin

#### Featured projects 👇
